# UI - Kokino Frontend

React-based user interface for visual agent orchestration.

## Phases

**Phase 1**: Canvas and drag-drop foundation
**Phase 2**: Agent library and connections
**Phase 3**: Mock orchestration with simulated data
**Phase 5**: Live integration with broker
**Phase 6**: Terminal integration (XTerm.js)

## Tech Stack

- React 18 + TypeScript
- Vite
- TailwindCSS
- React Flow (canvas)
- XTerm.js (terminals)
- Zustand (state)
